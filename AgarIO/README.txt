Мой аналог игры Чашка Петри(AgarIO) с кооперативом

# Подготовка
Установите системный Python3 или разверните venv
Установите требуемые модули/пакеты
pip install -r ./requirements.txt

Если сервер запускается на другом компьютере - в server.py изменить адрес сервера
Если игроков более 1 - каждому в client.py изменить my_name на желаемое имя

#Запуск
Запустите сервер
python server.py
Затем запустите клиент
python client.py
